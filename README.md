# Google Office List App
This app shows list of 10 Google offices around the world.

## Architecture that used in the app:
**Responsive Layout**
  *  using ListView for small screen and GridView for bigger screen over 700 pixel.

**Animated Navigation**
  *  navigate between `HomePage()` and `DetailPage()` using `Navigator.push()`.
  *  using `SlideTransition()` to create a sliding screen from below when navigating.

**Card form for each Item**
  *  using `Card()` widget to create each google office item and using `InkWell()` to create
     ripple effect when user touch the card.
